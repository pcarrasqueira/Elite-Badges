# Badge Image Database

Store project-owned badge images here when they should be served from this repository.

Use stable filenames, for example:

```text
database/images/source/remux.png
database/images/resolution/4k-ultra-hd.png
database/images/streaming/netflix.png
```

Once pushed to GitHub, the raw URL format is:

```text
https://raw.githubusercontent.com/pcarrasqueira/Elite-Badges/main/database/images/<section>/<file>
```
